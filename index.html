Proyectos Seleccionados

🚖 Movilidad urbana y productividad económica en Latinoamérica

Proyecto enfocado en analizar el impacto de la congestión vehicular sobre la productividad económica en ciudades latinoamericanas.

Objetivo

Identificar ciudades con mayor necesidad de inversión en infraestructura de transporte y movilidad urbana.

Desarrollo

  • Integración de datos de tráfico de TomTom.
  • Cruce de información con indicadores económicos de la OECD.
  • Limpieza y análisis de datos con Python y Pandas.
  • Visualización de patrones y correlaciones.

Herramientas utilizadas

Python | Pandas | Visualización de Datos

Resultado

Se identificó una relación entre el incremento en tiempos de congestión y la disminución del desempeño económico urbano.

📦 Análisis de embudo y retención para e-commerce

Proyecto orientado al análisis del comportamiento de usuarios dentro del proceso de compra en una plataforma de comercio electrónico.

Objetivo

Detectar puntos críticos de abandono en el embudo de conversión para mejorar la experiencia del usuario.

Desarrollo

  • Transformación y organización de tablas mediante SQL.
  • Análisis del flujo de usuarios en el proceso de compra.
  • Creación de visualizaciones para stakeholders.

Herramientas utilizadas

SQL | Visualización de Datos | Excel

Resultado

Se detectaron etapas con mayor fuga de usuarios, facilitando oportunidades de optimización en el proceso de compra.
